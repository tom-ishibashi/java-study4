# java-study4

## Description
Java学習用のアプリケーションです。  
アプリケーション仕様はSPECIFICATION.mdを参照。

## Requirements
- JDK 8
- Maven 3.x
- MySQL 5.x

## How to Build
```
mvn clean install -Dmaven.test.skip=true
```

## How to run
```
mvn exec:java
```
