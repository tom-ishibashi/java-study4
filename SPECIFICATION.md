# アプリケーション仕様
## 概要
- 図書館の情報を管理する風のアプリケーションである
- 図書館や本の情報を検索、登録、更新、削除できる
- 標準入力と標準出力で実現するJavaアプリケーションとする
- 情報はDBに永続化できる

# 動作イメージ

アプリケーションを実行すると、以下のような流れで処理が行われる。

## 操作を選択
1,2,3,4以外を入力したらアプリケーション終了。
```
操作を選択してください。
1.select 2.insert 3.update 4.delete
```

## 1を入力した場合は、検索
### 検索条件の入力
検索条件は未入力も可
```
検索条件を入力してください。
図書館:
本 ジャンル:
本 タイトル:
本 著者名:
本 出版社:
```

### 検索結果の表示
検索条件に合致する図書館と紐付く本の一覧を表示する。  
検索条件未入力の場合全件表示する。  
検索結果を表示するとアプリケーション終了。
```
図書館/ジャンル/本タイトル/本著者/出版社
A図書館/ミステリ/ミステリ本/乱歩さん/A出版
A図書館/エッセイ/エッセイ本/エッセーさん/C出版
B図書館/SF/SF本/ジョージさん/B出版
```

## 2を入力した場合は、登録
### 登録対象の選択
1,2,3以外を入力するとアプリケーション終了。
```
登録対象を選択してください。
1:図書館 2:本 3:ひも付き
```

### 登録内容の入力
#### 1.図書館を選択した場合
```
図書館名:

登録しました。
```
登録対象の選択に戻る。  
全て未入力の場合、処理をスキップして登録対象の選択に戻る。

#### 2.本を選択した場合
```
ジャンル:
タイトル:
価格:
著者:
出版社:

登録しました。
```
登録対象の選択に戻る。  
全て未入力の場合、処理をスキップして登録対象の選択に戻る。

#### 3.ひも付きを選択した場合
図書館を一覧表示する。
```
図書館一覧
No:図書館
1:図書館A
2:図書館B
```

本を一覧表示する。
```
本一覧
No:ジャンル/本タイトル/本著者/出版社
1:ミステリ/ミステリ本/乱歩さん/A出版
2:エッセイ/エッセイ本/エッセーさん/C出版
3:SF/SF本/ジョージさん/B出版
```

```
どの図書館にどの本をひも付けますか？
図書館No:
本No:

登録しました。
```
登録対象の選択に戻る。  
全て未入力の場合、処理をスキップして登録対象の選択に戻る。
